To generate the json file, get the list of language needs to be supported in the app.
Generate a locale.json file in the project with the list of supported locales.
The supported locale will be in the form of "language_country".

The example json file is as follows:

--------- JSON START ---------

[
  "en_US",
  "hi_IN",
  "zh_CN"
]

--------- JSON END ---------
